##현대사회의윤리적쟁점들(교수님)

##지리학입문(장규진)
일주일에 한번 듣는데 일찍 끝내주시고, 창의력에 따라 점수를 주십니다. 수업 잘 안 들어도 A+ 노릴 수 있습니다.

##한국전통문화의이해(허인욱)
보고서도 1~2장 정도로 작성하면 만점 주시고 무엇보다 피피티를 전부 올려주시기 때문에 공부하는 데 문제가 없습니다.
중간고사는 전통문화에 대한 자신의 생각을 중심으로, 기말고사는 수업 내용을 중심으로 내시는 편입니다.

##사고와표현(권혁명)
